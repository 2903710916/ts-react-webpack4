# ts-react-webpack4
typescript + react + webpack4 starter

## run
1. npm i
2. npm run dev

## joined
- webpack-4
- typescript
- react
- react-router-4
- component hot reload
- async component
- import .(s)css auto generate .(s)css.d.ts
- svg icon
