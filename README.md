# ts-react-webpack4

typescript + react + webpack4 starter

## 运行

1.  npm i
2.  npm run dev

## 功能(包)支持

-   webpack-4
-   typescript
-   react
-   react-router-4
-   Mobx
-   Antd
-   component hot reload
-   async component
-   import .(s)css auto generate .(s)css.d.ts
-   import svg icon as a component
-   create component folder by `npm run add`
